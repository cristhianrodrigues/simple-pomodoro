# simple-pomodoro

Este é um projeto de Temporizador Pomodoro construído usando React. O código importa os hooks useState e useEffect do React, bem como ícones da biblioteca react-icons. O temporizador possui várias configurações ajustáveis, incluindo tempo de foco, tempo de intervalo, tempo de intervalo longo e número de intervalos antes de um intervalo longo. O temporizador pode ser iniciado, pausado e redefinido usando as funções apropriadas. Um som de alarme é tocado quando o temporizador chega a zero. O temporizador também possui um modal para ajustar as configurações simplificando para o usuário.

Você pode experimentar este projeto em ação visitando o site https://simple-pomodoro-drab.vercel.app/
